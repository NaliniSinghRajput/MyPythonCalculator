# MyPythonCalculator
Hello there! I made my first calculator using Python
Here are the steps on how I actually made it : 
I created this calculator using the tools that I have been learning from the past 3 days

I firstly made a comment which tells us that we're making a calculator

Next I made several functions that are going to be working as the operations and added a and b as parameters for the function
the functions that i put on this calculator were Sum, Difference, Multiplication, Division, Power and a function (a + b) ** 2 

Next I made the main function which was the calculator and then I added an input for the user , where the user will be able to give a float value a and then another float value b

Next I used the if - elif - else statements to make the further loop where we can tell the program to perform the specific operation as inputted by the user and then print the result according to the function.

After this, I made a print statement that prints out all of the operations with specified numbers so that users can choose from them.

After this,next I gave the user a choice to choose from the operations using a number range.

And that was all!, this is how i made a very easy functionable calculator. I will be modifiying it later on according the needs, but it's a very basic calculator that works on plain and simple logic.
